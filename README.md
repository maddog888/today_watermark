# today_watermark
今日水印相机_纯HTML和JS水印实现方法

调配设备iPhone12

生成图片之后快速保存的方法
打开快捷指令
新建一个指令
-截屏
-裁剪-截屏-位置-自定-x：15-y：350-宽度：1140-高度：1485
-存储到最近项目
-生成之后可以叫Siri运行指令
